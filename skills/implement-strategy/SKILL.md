# Step 1
The user will point to the strategy file, and ask for creating a plan to implement a specific step in this strategy.

# Step 2
Create a plan based on the user's request, the [docs](../grow-docs/SKILL.md) and [testable-module](../testable-module/SKILL.md) strategies. [Grill the user](../grill-me/SKILL.md) for any missing details or clarifications needed to create a complete plan.

# Step 3
[Handoff the plan](../handoff-to-worktree/SKILL.md) to the user, preserving the [docs](../grow-docs/SKILL.md) and [testable-module](../testable-module/SKILL.md) strategies in the implementation steps. Also, add a final step to remove the implemented step from the strategy file.
