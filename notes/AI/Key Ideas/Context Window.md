Context Window is everything that the current conversation contains.

### Keep it small
AI results will worsen as the context window grows. 
Short and precise instructions are king.
Well [[Documentation and structure#^200e87|structured]] code is both smaller and more verbose.

### Best practice
#### Compact
Use /compact in the [[Agent & Workflow#^5985a1|CLI Agent]] to summarize the current context and keep it small.
#### Start new session
If your plan is ready, move to a new conversation to implement it to start with a fresh window.
